# Retail-Sales-Performance-Dashboard-End-to-End-Power-BI-Analytics-Project

**Objective**

The primary objective of this project was to build an interactive and visually-driven Power BI dashboard that enables stakeholders to monitor retail business performance across multiple dimensions such as:

- Sales Performance
- Profitability
- Product Performance
- Regional Trends
- Brand Analysis
- Store Type Contribution
- Year-over-Year Growth

The dashboard was designed to transform raw transactional retail data into actionable insights that support faster and data-driven business decisions.

**Problem Statement**

Retail businesses often deal with fragmented sales data spread across multiple regions, products, stores, and timelines.

Without a centralized analytical solution, it becomes difficult to:

- Track overall business performance in real time
- Identify top-performing products and brands
- Compare profitability across regions and states
- Analyze yearly sales trends
- Monitor quantity movement and profit margins
- Enable leadership teams to make quick strategic decisions

The challenge was to consolidate this data into a single, interactive reporting solution with meaningful KPIs and intuitive visual storytelling.

**Process Followed**

**1. Data Collection & Preparation**
Imported retail sales datasets into Power BI
Performed data cleaning and transformation using Power Query
Handled null values, inconsistencies, and data formatting issues
Created a proper data model with relationships between fact and dimension tables

**2. Data Modeling**
Implemented a star schema structure using:

- Fact Tables
- Calendar Table
- Product Dimension
- Customer Dimension
- Store Dimension

This helped improve report scalability and query performance.

**3. DAX Measures Creation**

Built custom DAX measures for:

- Total Sales
- Total Profit
- Total Quantity
- Profit Margin
- Year-wise Filtering
- Comparative Analysis

Focused on reusable and optimized DAX logic for better performance.

**4. Dashboard Development**

Designed an executive-style dashboard featuring:

- KPI Cards
- Trend Analysis
- Product Rankings
- Regional Performance Mapping
- Store Type Distribution
- Brand Performance Tables
- Interactive Slicers for Year Filtering

**5. UI/UX & Visualization Design**

Special attention was given to:

- Dark theme dashboard aesthetics
- Consistent color palette
- Business-friendly visual hierarchy
- Clean navigation and readability
- Interactive storytelling

**Key Insights Delivered**
- Identified top-performing products by sales
- Analyzed profit contribution across regions
- Compared store-type sales distribution
- Tracked monthly sales trends
- Evaluated state-wise business performance
- Measured operational profitability using margin KPIs

**Desirable Outcome**

The final solution provides:

✅ A centralized retail analytics dashboard
✅ Faster decision-making through real-time insights
✅ Improved visibility into sales and profitability
✅ Better product and regional performance tracking
✅ Scalable reporting architecture for future expansion
✅ Enhanced executive reporting experience
